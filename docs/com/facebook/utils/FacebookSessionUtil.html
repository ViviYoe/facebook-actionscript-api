Classpublic class FacebookSessionUtil
InheritanceFacebookSessionUtil flash.events.EventDispatcher

The FacebookSessionUtil class provides a convenient way to create or continue a Facebook session. This class handles many of the tasks that you would otherwise have to handle manually. The FacebookSessionUtil class constructor:
  • Checks whether an existing session is stored as a SharedObject on the user's computer and uses that session key if it exists.
  • Checks whether the Facebook server provided a new session key and gives that session key precedence over the SharedObject session key.
  • Checks whether the Facebook server sends a session secret to use in place of your application secret and uses the session secret if it exists.
  • Attempts to determine what type of session you are using, either a DesktopSession, a WebSession, or a JSSession (JavaScript Bridge Session). If it cannot determine the type of session, it uses a DesktopSession.
  • Creates a session of the type determined in the previous step and stores the session in the public property named activeSession.
  • Creates an instance of the Facebook class and stores the instance in a public property named facebook.
  • Starts a session of the appropriate type.

Constructor detail
FacebookSessionUtil()constructor
public function FacebookSessionUtil(api_key:String, secret:String, loaderInfo:LoaderInfo)

The constructor creates a new session of the appropriate type. See the FacebookSessionUtil class description for detailed information about the constructor.

Parameters
api_key:String — Your application's API key.
 
secret:String — Your application's secret key. If this parameter is passed a value of null, the constructor looks for a special session secret stored in the fb_sig_ss property of the loaderInfo object. For web sessions, even if you pass a non-null value for this parameter, the constructor will always look for a session secret and use that instead of the value that you pass for this parameter.
 
loaderInfo:LoaderInfo — An object of type LoaderInfo that provides information about the loaded SWF file.
