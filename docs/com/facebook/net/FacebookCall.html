handleResult()method 
facebook_internal function handleResult(result:FacebookData):void

Called from the IFacebookCallDelegate when the communication has been completed. This method will determine the success, parse out any errors, call the handleSuccess method on successful calls (so that extended classes can easily handle the success of the call) and inform any listeners/callbacks that the call is complete.

Parameters
result:FacebookData — Object the result object provided from the Facebook REST services
initialize()method 
facebook_internal function initialize():void

Initialize the facebook call. This will normally be used to convert class variables into arg values. Because a call can be used over and over if desired this method will be called so that child classes can re-initialize the .arg values.
