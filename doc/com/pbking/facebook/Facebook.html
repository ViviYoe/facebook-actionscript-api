startDesktopSession()method 
public function startDesktopSession(api_key:String, secret:String, infinite_session_key:String = ""):void

Start a session for a desktop based application. For this you need to know and pass the application api_key and secret. The user will be prompted to login to their Facebook page after which you should call validateDesktopSession(). Alternately you could pass in an infinite_session_key which will authenticate immediately (without navigating to the user login page).

Parameters
api_key:String
 
secret:String
 
infinite_session_key:String (default = "")

startWebSession()method 
public function startWebSession(loaderInfo:LoaderInfo, api_key:String = null, secret:String = null):void

Start a session for a web-based application. You can choose not to pass in your secret if you plan on using a proxy server for your facebook calls. If your api_key is set here and is also passed in via flashvars then that that value will be ignored.
